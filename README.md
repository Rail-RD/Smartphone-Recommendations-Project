#  Smartphone Recommendation System

## Описание проекта
Интеллектуальная система подбора смартфонов на основе технических характеристик и предпочтений пользователя. Проект решает проблему выбора устройства в условиях перенасыщенного рынка, используя алгоритмы машинного обучения.

## Основной функционал
* **Content-Based Filtering:** Система анализирует параметры выбранного устройства (процессор, ОЗУ, камера, цена) и предлагает максимально похожие аналоги.
* **Интерактивный подбор:** Фильтрация по критическим параметрам (бюджет, тип матрицы, объем батареи).
* **Визуализация сходства:** Наглядное сравнение характеристик рекомендуемых моделей.

## Стек технологий
* **Язык:** Python
* **Библиотеки:** * `Pandas` — обработка и очистка данных.
    * `Scikit-learn` — реализация алгоритмов (например, Cosine Similarity для поиска схожих устройств).
    * `Seaborn / Matplotlib` — исследовательский анализ данных (EDA).
    * `Streamlit` (если использовали) — создание веб-интерфейса.

## Этапы разработки
1. **Сбор и предобработка данных:** Очистка датасета от дублей, обработка выбросов в ценах, нормализация числовых признаков (Scaling).
2. **EDA (Исследовательский анализ):** Поиск корреляций между ценой и производительностью, анализ распределения брендов.
3. **Разработка модели:** Реализация алгоритма вычисления сходства между объектами.
4. **Интерфейс:** Разработка UI для ввода параметров пользователя и отображения результатов.

## Запуск проекта 
1.	Выполните команду: pip install pandas scikit-learn fastapi uvicorn jinja2.
2.	Запуск проекта: main.py. Это создаст файлы phone_model.pkl, scaler.pkl и processed_data.pkl.
3.	Ввести в минал: uvicorn main:app --reload.
4.	Нажать на в терминале http://127.0.0.1:8000 для открытия приложения в браузере.
